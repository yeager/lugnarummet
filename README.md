# Lugna Rummet

Sensory regulation and calming strategies for autism and ADHD.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install lugnarummet
```

### Fedora/RPM
```bash
sudo dnf install lugnarummet
```
